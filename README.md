## Быстрый старт

<br />

1. Склонировать проект на свой компьютер

```bash
git clone https://github.com/SergeyKazarinov/yandex.practicum.git
```

2. Перейти в папку с проектом и установить зависимости в проекте

```bash
cd yandex.practicum
```
```bash
npm install
```

3. Запустить проект

```bash
npm start
```
